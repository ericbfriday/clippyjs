# Welcome to ClippyJS React Storybook

## What is ClippyJS?

ClippyJS brings back the classic Microsoft Office assistant characters as interactive React components. Built with modern TypeScript and React 19, it provides a production-ready, type-safe API for adding nostalgic agent assistants to your applications.

## Quick Start

### Installation

```bash
npm install @clippyjs/react
```

### Basic Usage

```tsx
import { ClippyProvider, useAgent } from '@clippyjs/react';

function App() {
  return (
    <ClippyProvider>
      <MyComponent />
    </ClippyProvider>
  );
}

function MyComponent() {
  const { load, speak } = useAgent('Clippy');

  return (
    <button onClick={async () => {
      await load();
      await speak('Hello! Welcome to our website!');
    }}>
      Show Clippy
    </button>
  );
}
```
